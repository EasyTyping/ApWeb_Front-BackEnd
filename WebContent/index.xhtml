APLICACIÓN BASADA EN EL PATRÓN MVC

Para el desarrollo se ha utilizado Eclipse Neon 3 (4.6) y JDK 1.8_u161 (jre 1.8_161). Además se han utilizado los siguientes Frameworks y librerías:

  • HIBERNATE 4.1.4 Y SPRING 4.3.9
  • JSF 2.2 Y ICEFACES 4

El nivel del LOG4J se ha dejado a TRACE para obtener una información detallada por consola (puede cambiarlo si lo desea en el propeties). Del mismo modo, puede consultar el fichero c:/java_bejarano/log.txt.

Se ha incluido un DTO simplificado y un DAO para la tabla Clientes

Tanto al borrar o modificar un pedido, como un artículo, se borran y modifican sus líneas asociadas.

Para los mensajes al usuario se ha utilizado el objeto FacesMessage.Se ha utilizado una plantilla para el menú y un CSS para el posicionamiento.
